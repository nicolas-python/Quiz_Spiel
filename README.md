# Quiz Spiel

Ein Python-Quizspiel, bei dem der Spieler Fragen beantworten muss.Der Score wird gespeichert.

## Funktionen

- Spieler erstellen
- Spieler auswählen
- Quizfragen beantworten
- Score speichern
- Score anzeigen
- Spiel beenden

## Nutzung

Starte das Programm mit:

Python quiz_spiel.py

Danach erscheint ein Menü mit folgenden Optionen:Spieler erstellen,Spieler wählen,Spielen,Score anzeigen,Beenden

## Spielablauf

1. Zuerst einen Spieler erstellen oder Spieler auswählen.
2. Danach Spielen auswählen.
3. Die Fragen werden angezeigt und der Spieler gibt die Nummer der richtigen Antwort ein.
4. Für jede richtige Antwort erhält der Spieler einen Punkt.
5. Am Ende wird der Score angezeigt und gespeichert.

## Beispiel Frage 

Wähle eine Option: 3

Was ist Python?
1 : Ein Programm
2 : Ein Nahrungsergänzungsmittel
3 : Eine Automarke
4 : Ein Promi
